# 📈 Live Portfolio Risk & Optimization Dashboard

A real-time financial analytics dashboard built with **Python** and **Streamlit** that allows users to analyze portfolios using live market data.

---

## 🚀 Features

- 📡 Live stock data using Yahoo Finance
- 📊 Portfolio risk metrics (VaR, Expected Shortfall)
- 🔗 Correlation matrix
- 📉 Loss distribution visualization
- 📊 PCA (Principal Component Analysis)
- ⚙️ Portfolio optimization
- 🤖 Machine learning predictions
- 📈 Sharpe ratio calculation

---

## 🧠 What You Can Learn

This dashboard helps you understand:

- How much your portfolio can lose in extreme scenarios
- How diversified your assets are
- What factors are driving your risk
- Whether your portfolio is efficiently balanced

---

## 🌐 Live Usage

Run locally or deploy on Streamlit Cloud.

---

## ⚙️ Installation

```bash
git clone https://github.com/your-repo/live-portfolio-dashboard.git
cd live-portfolio-dashboard
pip install -r requirements.txt
streamlit run app.py
```
